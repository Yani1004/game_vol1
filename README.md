# Наследство+ / Heritage Hunt

Android мобилна игра за откриване на български културно-исторически обекти чрез карта, GPS, точки, дневни предизвикателства и отбори.

## Какво покрива проектът

- Интерактивно Android приложение с интерфейс на български по подразбиране.
- Регистрация, вход и локален профил.
- Google Maps екран с реални координати на обекти в България.
- GPS откриване в радиус 250 метра.
- Стартиране на камера от екрана с картата.
- Дневно notification напомняне.
- История на посещенията с дата, точки и историческа информация.
- Дневно предизвикателство с бонус точки.
- Отбори с код за покана, отворен/затворен достъп, заявки и класация.
- Документация за защита в `docs/PROJECT_DOCUMENTATION.md`.
- Бележки и структура за 5-минутна презентация в `docs/DEFENSE_NOTES.md`.

## Технологии

- Kotlin
- Android SDK
- XML layouts
- Google Maps SDK
- Google Play Services Location
- SharedPreferences
- Android Notifications

## Стартиране

```powershell
./gradlew.bat assembleDebug
```

Проектът е локален прототип, подходящ за курсова защита. За реална онлайн версия следваща стъпка е Firebase Authentication и Firestore.
